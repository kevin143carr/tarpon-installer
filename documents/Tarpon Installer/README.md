# Tarpon Installer Slidev Deck

This folder contains a Slidev presentation about the current Tarpon Installer feature set.

To run it locally:

- `npm install`
- `npm run dev`
- open <http://localhost:3030>

To build static output:

- `npm run build`

To export presentation output:

- `npm run export`

Edit [slides.md](./slides.md) to update the deck.
